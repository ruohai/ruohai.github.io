Linux下实现USB设备自动挂载

Jul 15, 2023
1 min read

给玩客云刷的armbian默认不支持usb设备自动挂载，接下去就是找实现方案了，有udev/udevil/udiskie/..。简单了解了一下，最后选择用udevil/devmon，因为casaos选择的就是udevil/devmon，而且装好以后不需要写脚本，😄。

安装udevil/devmon

apt install udevil -y

配置文件的默认路径是/etc/udevil/udevil.conf，默认会把usb设备挂载到/media/user/目录下（这里的user在后面的命令中体现）。要调整挂载路径，需要在udevil.conf中修改。

接下去要启动udevil/devmon服务。注意命令中的user，这里对应的是你的linux系统的用户账户，如果你用的是root账户，那就是devmon@root，最后usb设备的挂载点就是/media/root/。你也可以指定挂载到其它存在的用户账户，比如devmon@jack，那挂载点就是/media/jack/。

systemctl start devmon@user.service

将devmon添加到开机启动服务。

systemctl enable devmon@user.service

完成以上配置以后，就不需要再做任何配置了，真无感使用。

但我实际使用中发现有一点点小问题，系统重启以后，sd卡槽中的sd存储卡不能自动挂载，需要有一个重新插拔的动作以后才能识别。
