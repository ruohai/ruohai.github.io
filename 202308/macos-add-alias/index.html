MacOS下给终端添加alias

Aug 23, 2023
1 min read
MacOS alias

目前macos默认shell是zsh，简单记录下给zsh添加自定义命令。

打开终端，进入用户目录。

cd ~

我的机器默认没有.zshrc的配置文件，新建一个。

touch .zshrc

然后在.zshrc中添加alias。

alias ll='ls -l'
alias la='ls -al'

保存退出，然后载入配置文件

source .zshrc

到这里就完成配置了～
